UE 4 - Gérer des données de l'information

Gestion des données de Parcoursup

Au sein de ce projet nous devions recréer une base de données similaire à celle de Parcoursup, puis ensuite manipuler ces données afin d'obtenir les résultats attendus. Ce projet m'a permis de mieux appréhender la conception d'une base de données et les différentes manières de l'interroger afin d'obtenir les résultats souhaités.

  • Création
  • Peuplement

Création

PostgreSQL

Depuis un diagramme UML ainsi qu'un fichier CSV, nous devions créer les fondations de notre base de données.

Peuplement

PostgreSQL

En partant d'un fichier de création sain, nous avions ensuite pour mission de peupler la base de données à l'aide des données du fichier CSV.
